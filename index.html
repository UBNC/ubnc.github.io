Models

We provide model weights for multiple BioNER tools in two formats. We give the weights as BLABLA file, that can be imported by BLABLA ML toolkit. In addition we provide Docker containers, which can be used as out of the box taggers. An evaluation of the tagger performance can be found in our publication.
